# Computer Vision Course

This directory contains instructional notebooks introducing image representation, NumPy-based image processing, deep-learning concepts for computer vision, and object-detection assessment tasks.

The materials are intended for classroom demonstrations, guided laboratory exercises, technical assessment, and independent practice.

## Learning Objectives

After completing these notebooks, students should be able to:

- explain how digital images are represented as numerical arrays;
- create and manipulate NumPy arrays;
- inspect image dimensions, shapes, and data types;
- access and modify image pixels;
- apply indexing and slicing to image data;
- display grayscale and colour images;
- explain the role of convolutional neural networks in computer vision;
- identify the stages of an image-classification workflow;
- distinguish image classification from object detection;
- interpret bounding boxes, object classes, and confidence scores;
- evaluate computer-vision outputs critically;
- identify limitations related to data quality and model generalization.

## Notebook Overview

| Notebook | Main topic |
|---|---|
| `Introduction_to_Numpy_and_Image_Basics.ipynb` | NumPy fundamentals and digital-image representation |
| `DL_CV_Assessment.ipynb` | Assessment of deep-learning and computer-vision concepts |
| `Detection_Assessment.ipynb` | Assessment of object-detection concepts and workflows |
