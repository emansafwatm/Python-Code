# AI & Machine Learning Teaching Portfolio

Welcome to my **AI & Machine Learning Teaching Portfolio**.

This repository contains a structured collection of educational notebooks, practical projects, and reproducible examples covering Data Science, Machine Learning, Deep Learning, Natural Language Processing, Large Language Models, and related AI topics.

The goal is to provide clear, hands-on learning material while demonstrating practical implementation using modern Python AI libraries.
