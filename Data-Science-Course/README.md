# 📊 Data Science Course

A hands-on collection of Jupyter notebooks covering the essential concepts of modern data science using Python.

This course is part of the **AI-ML-Teaching-Portfolio** and follows a progressive learning path from data preparation to complete analytical workflows.

---

## 📚 Course Contents

| Notebook | Topic |
|----------|------|
| 01 | Data Cleaning & Missing Values |
| 02 | Exploratory Data Analysis (EDA) |
| 03 | Data Visualization |
| 04 | Descriptive Statistics |
| 05 | Correlation & Association |
| 06 | Grouping, Aggregation & Pivot Tables |
| 07 | Statistical Hypothesis Testing |
| 08 | End-to-End Data Analysis Case Study |

## ▶️ Getting Started

Clone the repository

```bash
git clone https://github.com/emansafwatm/AI-ML-Teaching-Portfolio.git
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

**Windows**

```powershell
.\.venv\Scripts\Activate.ps1
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter

```bash
jupyter notebook
```

---

## 📖 Recommended Learning Order

Run the notebooks sequentially:

**01 → 02 → 03 → 04 → 05 → 06 → 07 → 08**

Each notebook builds upon concepts introduced in the previous one.
