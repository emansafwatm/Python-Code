# Data Science Course Notebooks

The notebooks provide practical introductions to data cleaning, exploratory analysis, visualization, descriptive statistics, correlation, and applied analysis using Python.

## Suggested Study Sequence

1. `Data_Cleaning_and_Missing_Values.ipynb`
2. `Exploratory_Data_Analysis.ipynb`
3. `03_Data_Visualization_with_Matplotlib.ipynb`
4. `04_Descriptive_Statistics.ipynb`
5. `05_Correlation_and_Association.ipynb`
6. `Correlation.ipynb`
7. `US_Presidential_Election_Analysis.ipynb`

## Notebook Topics

| Notebook | Main topic |
|---|---|
| `Data_Cleaning_and_Missing_Values.ipynb` | Missing values, duplicates, invalid records, type conversion, validation, and cleaning documentation |
| `Exploratory_Data_Analysis.ipynb` | Dataset inspection, distributions, grouped summaries, correlation screening, outlier review, and interpretation |
| `03_Data_Visualization_with_Matplotlib.ipynb` | Line, bar, histogram, scatter, box, and pie charts; subplots; annotations; chart selection; misleading graphs |
| `04_Descriptive_Statistics.ipynb` | Mean, median, mode, range, variance, standard deviation, percentiles, quartiles, IQR, outliers, and skewness |
| `05_Correlation_and_Association.ipynb` | Covariance, Pearson and Spearman correlation, matrices, heatmaps, scatter plots, subgroup analysis, outlier effects, and causation limits |
| `Correlation.ipynb` | Existing correlation lesson retained for review or future consolidation |
| `US_Presidential_Election_Analysis.ipynb` | Applied exploratory analysis of historical election data |

## Dataset Used in Lessons 04 and 05

`../datasets/employee_statistics.csv`

The dataset contains 500 synthetic employee records and includes numerical, categorical, and date variables.
